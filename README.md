# Project: GameOfTheTower
WIP tower defence game with roguelike influences, made for mobile platforms
----
## What's this?
- Defend your cores from humongus amounts of enemies
- buy tiles and build your weapons on them <sub>*No real money involved*</sub>
- Search new weapons across the map
- Use tactic to make the best out of what you find
- **Break Everything**
----
## Someone said *Randomness*?
- Randomly Generated maps every time
- Randomly placed cores and enemy spawners
- Random item pools for powerups with random positions
